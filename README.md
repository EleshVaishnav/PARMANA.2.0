# Parmana AI 🤖

Parmana is an open-source, API-agnostic, agentic AI. It supports multiple LLM providers (OpenAI, Anthropic, Gemini, Groq), uses tools/skills, has long-term memory, and connects directly to channels like Telegram and WhatsApp.

## 🚀 Features
- **Bring Your Own API**: Supports any major LLM provider.
- **Multilingual**: Understands and speaks all languages inherently.
- **Agentic Skills**: Can search the web, execute code, and solve math.
- **Memory**: Remembers user facts and previous conversations.
- **Multi-Channel**: Talk to Parmana via Telegram or WhatsApp.
